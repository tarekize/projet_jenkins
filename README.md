# TP Jenkins - ICDE848

Projet Maven pour le TP Jenkins avec intégration CI/CD complète.

## Configuration

- **Langage** : Java 21
- **Build Tool** : Maven 3
- **Tests** : JUnit 5 + Mockito
- **Qualité Codes** : Checkstyle, PMD, SpotBugs, JaCoCo

## Pipeline Jenkinss test

Le pipeline automatisé inclut :
1. **Checkout** - Récupération du code depuis GitHub
2. **Build** - Compilation du projet
3. **Unit Tests** - Exécution des tests unitaires
4. **Checkstyle** - Vérification des conventions de code

Dernier test d'automatisation CI/CD via Webhook le 13/04/2026.
